# Computer-Architecture-Semester-Project-Instruction-Mix-Analyzer-
Inside this repository will be the source code and test code to an Instruction Mix Analyzer. This analyzer will be able to take in a list of instructions and return the instruction types. This project uses Python and VS code to make the code.

To accomplish this task, AI bots (Claude and DeepSeek were used to help establish and understand the code).
